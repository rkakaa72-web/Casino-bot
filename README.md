# Telegram Game Bot

Игровой Telegram-бот с балансом, работой и мини-играми:

- `Верх/Низ`
- `Рулетка`
- `Мины`
- `Работа -> Грузчик`

## Запуск

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Создать `.env`:

```bash
cp .env.example .env
```

3. Указать токен бота в `.env`

4. Запустить:

```bash
python3 bot.py
```
